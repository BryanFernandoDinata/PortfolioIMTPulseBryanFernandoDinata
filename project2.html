CAFE!

About This Project


This project is an ICT (Information Communication and Technology) project. It is about making 3D Japanese Styled Cozy Cafe using Blender Engine. This project was a task for all the 12B Intan Permata Hati Student, so each of 12B Student made a donut 3D Japanese Styled Cozy Cafe to get good grades. But the one that I show in this page is my own 3D model.


Doing this project was really fun. This is certainly not my first project to make a 3D model, but it was my first time making a 3D Japanese Styled Cozy Cafe using Blender Engine. So, it was a new experience for me.


In this project I learned how to bevel, subdivide, solidify, and modify geometries. This is because I need to bevel, subdivide, solidify and modify the geometries a lot in this project.
