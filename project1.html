DONUTS!

About This Project


This project is an ICT (Information Communication and Technology) project. It is about making Donuts 3D model using Blender Engine. This project was a task for all the 12B Intan Permata Hati Student, so each of 12B Student made a donut 3D model to get good grades. But the one that I show in this page is my own 3D model.


Doing this project was really fun. This is certainly not my first project to make a 3D model, but it was my first time making a 3D donut model using Blender Engine. So, it was a new experience for me.


In this project I learned how to texture a 3D object in Blender Engine. This is because, the 3D object needs to be textured to have a texture and good colors. Not only that, but I also learned to create a good material in blender.
